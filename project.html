Symbols

@ { & * " #

1. When we need to include trademark or other symbols into copy, include the without the <sup> tag.
2. When selecting a symbol as part of your design, make sure the element enhances the idea and is ib harmony with copy within the asset.
For example:
Use the number symbol to talk about a new campaign and/or promore a hashtag.
Use the forward slash symbol when talking about a new webpage or a link.
